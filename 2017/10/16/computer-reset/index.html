Computer Reset Notes

Oct 16, 2017 • notes

It’s a good practice to reset your heavily used machine every once in a while. Usually, I wipe my mac at least once a year. I like having a fresh start without any clutter on my laptop (although I could restore it from the latest backup, I never want to).

With all the modern tools, wiping and reinstalling the OS is pretty simple - just keep clicking the “Next” button and in less than an hour it’s done. As a result, you get a “brand new” computer, as if you just came back from the Apple store with a new laptop.

What is super painful, though, is setting up the tools that make our lives so much easier. I’m finally making a list of things I always go searching for after my annual wipe.

Sublime

Conda

Create environment.

conda create --name bunnies python=3.5

Check what environments are installed.

conda info --envs

Git

Don’t forget to set your git email account properly, or your commits will show up under your computer’s local name.

git config --global user.email "git@email.com"

Bash

To change the shell prompt, modify the variable PS1. The special character for the current directory is \w.

$ PS1="[\w ] $ "
